# Rocket in Space Simulator

This is a project focused on developing an accurate simulation of a rocket being launched from one planet to another. The goal is to ofcourse learn graphics, numerical simulations, computational physics and rocket science. The project is divided into phases with each phase going one level deeper. 

## Phase 1
The goal of this phase is to setup the basic scaffolding of the solar system. We shall setup the interfaces for the planets and visualize them on a simple opengl surface. The graphics will be in 3d with only plain sphere. 

Celestial bodies to be included:

    - The moon of the earth
    - The 9 planets
    - The Sun
We use a coordinate system centered on the sun to do this and an axis aligned with the elliptical plane of the Earth.
